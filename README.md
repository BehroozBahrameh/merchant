# Merchant
A sample API to recive a signal and process it!

## Tech Stach
- Java 17
- Spring Boot

## How to run
open termain and excute `run.ps1`

## Endpoint
to send signal call `~/signals/[signal]`, for example `http://localhost:8080/signals/1`
